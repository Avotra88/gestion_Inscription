# Gestion des Inscriptions

## Description
Ce projet est une application de gestion des inscriptions dans un système. Elle permet de suivre les ajouts, modifications et suppressions d'inscriptions.

## Fonctionnalités
- Suivi des actions effectuées sur les inscriptions (ajout, modification, suppression).
- Interface d'administration pour gérer les utilisateurs et les inscriptions.
- Statistiques sur les actions effectuées dans le système.
